# Портфолио сайт

Простой статический сайт портфолио на HTML/CSS/JS.

## Как использовать

- Открой `index.html` в браузере.
- Меняй тексты в `index.html`.
- Меняй цвета и шрифты через переменные в `styles.css`.
- Добавляй новые секции, карточки проектов и услуги.

## Что уже сделано

- Адаптивный макет
- Хедер с мобильным меню
- Секции: проекты, услуги, обо мне, контакты
- Анимации появления при прокрутке
- Настраиваемая тема
